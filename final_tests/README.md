## Итоговые тесты

В данной директории содержатся примеры для приемочных тестов вашего проекта. 
Разработанный анализатор должен поддерживать анализ данных тестов. 

Реализуйте набор тестов, демонстрирующий успешность анализа каждого из предложенных примеров. 
Тесты должны включать в себя вывод всех итоговых состояний, найденных анализатором, в человеко-читаемом формате.
