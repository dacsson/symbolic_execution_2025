# Домашнее задание 4: Внутрипроцедурный анализ основных конструкций

## Цель
Реализовать основной цикл анализатора, а также интерпретацию основных языковых конструкций.

## Задания

### Задание 4.1: Реализация основного цикла анализатора

В файле `analyser.go` необходимо реализовать функцию `Analyse(source string, functionName string) []Interpreter` таким образом,
чтобы она выполняла основной цикл работы анализатора — доставала из `Analyzer.StatesQueue` при помощи `Analyser.PathSelector` следующее состояние
интерпретатора для исполнения, вызывала шаг динамической интерпретации, и складывала все получившиеся состояния обратно в приоритетную очередь
до тех пор, пока не сработает глобальная стратегия остановки (стратегию остановки выберите и реализуйте самостоятельно).

### Задание 4.2*: Реализация Path Selectors

При желании, реализуйте дополнительные стратегии выбора пути в файле `path_selector.go`.

### Задание 4.3: Интерпретация основных языковых конструкций

В файле `dynamic_interpreter.go` реализуйте следующие методы:

* `interpretDynamically(element ssa.Instruction) []Interpreter`, который по заданной ssa инструкции проводит шаг интерпретации
и возвращает полученный набор новых состояний
* `resolveExpression(value ssa.Value) symbolic.SymbolicExpression`, который по заданному expression получает символьное выражение.

Интерпретация должна поддерживать все основные языковые конструкции - ветвления, примитивы, бинарные и унарные операции, циклы, ...
За исключением вызовов функций, операций с массивами, объектами и указателями. На данном этапе требуется внутрипроцедурный анализ.
