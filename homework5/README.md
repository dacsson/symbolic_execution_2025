# Домашнее задание 5: Межпроцедурный анализ оставшихся конструкций

## Цель
Реализовать межпроцедурный анализ, а также интерпретацию оставшихся языковых конструкций. 
В том числе добавить поддержку вызовов функций, операций с массивами, объектами и указателями.

## Задания

### Задание 5.1: Интерпретация оставшихся языковых конструкций

В файле `dynamic_interpreter.go` необходимо реализовать интерпретацию оставшихся языковых конструкций, необходимых для поддержки тестового набора.
На данном этапе необходимо также поддержать вызовы функций (Задание 5.2), операции с массивами, объектами и указателями.
Для этих целей используйте `symbolic_memory.go`.

### Задание 5.2: Поддержка межпроцедурного анализа

Необходимо поддержать межпроцедурный анализ в интерпретаторе. Используйте для этого `Interpreter.CallStack`. 
Подумайте о том, как работать с рекурсивными и взаимно-рекурсивными вызовами.
